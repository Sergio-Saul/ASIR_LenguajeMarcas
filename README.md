# Lenguaje de Marcas - Tareas Evaluativas

## Autor
Me llamo **Sergio Saul Luis Doble**.

## Descripción del repositorio
Este repositorio lo utilizaré para adjuntar mis tareas evaluativas de la asignatura de **Lenguaje de Marcas**.

Aquí iré subiendo los distintos archivos y prácticas que vaya realizando durante el curso, principalmente trabajos desarrollados con **HTML** y **CSS**.

## Sobre la realización de los trabajos
Los archivos HTML y CSS de este repositorio han sido realizados por mí, aplicando las etiquetas, estructuras y contenidos que se han explicado en clase.

Para algunas dudas puntuales he hecho preguntas básicas a la inteligencia artificial como apoyo, sobre todo para entender mejor ciertos errores o pequeños detalles. Aun así, he intentado en todo momento que la IA no hiciera cambios o ajustes que no se hayan visto en clase, ya que mi intención es **aprender el proceso paso a paso** y comprender lo que estoy haciendo.

## Objetivo
El objetivo de este repositorio es guardar y organizar mis prácticas, mostrar mi evolución en la asignatura y mantener un registro de los ejercicios evaluativos que vaya realizando.

## Tecnologías usadas
- HTML
- CSS

## Nota
Este repositorio tiene un fin principalmente académico y de aprendizaje.
